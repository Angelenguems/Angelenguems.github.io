## 2- SMS Spam Detection — NLP Classification Pipeline

**Objectif :** Concevoir un pipeline complet de détection de spam basé sur le NLP.

### Pipeline

1. Nettoyage et normalisation du texte
2. Vectorisation TF-IDF
3. Entraînement de modèles supervisés
4. Validation croisée et évaluation (Accuracy, Precision, Recall, F1-score)

### Performance

- Optimisation des hyperparamètres
- Comparaison de plusieurs classifieurs
- Analyse des métriques de performance

### Technologies

Scikit-Learn, NLTK, Pandas
